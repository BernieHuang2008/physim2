# 变量监视器
变量监视器可以使抽象的数据可视化。最基础的用法是绘制一些物理量随时间的变化曲线（如位置、速度、能量等）。
你还可以通过表达式的灵活性，构造出更加自定义的图表。e.g. 将X轴设置为某个对象的X坐标、Y轴设置为Y坐标，从而实时描绘出运动轨迹。

# 反作用力
注意反作用力。
如，在万有引力的例子中，A对B的引力作用力是 `- G * mass_A * mass_B / r^2`，而B对A的反作用力是 `G * mass_A * mass_B / r^2`。如果不考虑反作用力，那么每个物体就会受到双倍的引力。因此对于万有引力，我们可以在表达式中乘以 `mass / (mass + master_mass)` 来抵消掉一些引力，使得两个物体的两个作用力和反作用力的总和等于 `G * mass_A * mass_B / r^2`，而不是 `2 * G * mass_A * mass_B / r^2`。
又比如，在全局重力的例子中，重力必须绑定在anchor上，否则所有物体的重力产生的反作用力如果作用在一个有质量的物体上，就会导致该物体受到一堆反向重力，导致系统失控。
